1. Тут самые главные это две папки:
	1. client
	2. server

2. client - папка для хранения файлов, связанные с frontend
3. server - папка связанная с backand(База данных, фото на сайте)


код в папке клиент находится по пути client/src тут весь самый важный код для сайта
код в папке сервер находится сразу же после открытия server



НАЧНЕМ с client

1 папка AdminPanel:

	Тут находятся 2 файла для админ панели, это страницы для создания путевок на сайт

2 папка components:

	Тут находятся страницы, которые чаще всего используются на сайте

3 папка http:

	AviaApi - файл для запросов на сервер для путевок
	index - файл для создания адресов и путей
	userApi - файл для запросов на сервер для пользователя (Ну тут названия сами за себя говорят)

4 папка img тупо для фоток

5 папка pages: тут все страницы, на которые можно зайти: главная, регистрация и т.д

6 папка store: 

	DeviceStore - для создания стран, путевок, в массив, что бы они вообще показывались на сайте
	UserStore - тоже самое, только для пользователя, что бы сайт видел, зарегестрирован он или нет и т.д

7 папка styles: папка для стилей сайта. Надеюсь CSS ты помнишь

8 папка utils: Там один файл, в нем лежат все константы для роутинга по сайту(Пример: "https://vk.com/im" ---- https://vk.com"""""/im"""" - вот это то, что там прописано)

App.js - один из самых главных файлов, он базовый, там просто компоненты лежат
index.css - стили для сайта
index.js - САМЫЙ главный файл во всем проекте, с него все начинается

Потом server:

	controller:
		aviaController - файл для добавления путевок в базу данных
		typeController - файл для добавления стран в базу данных
		userController - файл для добавления пользователей в базу данных
	
	error - обработчик ошибок на сайте
	middleware - кадждый файл обрабатывает и ищет ошибки на сайте (Пример:  у пользователя неверен email, он это напишет)
	models - описана модель базы даных
	routes - post и get запросы с базы данных на сайт для пользователя, путевок, стран(Там 3 файла, все понятно какой где)
	static - туда добавляются фотки, которые ты загружаешь при создании путевок

.env - описание переменных с паролями к  базе данных и т.д
db.js - подключение к базе данных
index.js - главный файл, подключает базу данных, создает ее, стартует сам сервер
